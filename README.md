# forest

## サイト概要
日々の内容を投稿するブログサイト

### サイトテーマ
普段、誰にも言えない気持ちや日々あったどうでもいいことなどを投稿するサイト

### テーマを選んだ理由
誰でも利用したことのあるブログサービスを作ってみたい。
（ameba blogやTwitter)をイメージ。

### ターゲットユーザ
老若男女誰でも

### 主な利用シーン
日常で感情が動く出来事があった時に投稿する。
